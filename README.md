# Personal Portfolio Website

A modern and responsive personal portfolio website developed using HTML and CSS. This portfolio showcases my education, technical skills, projects, and contact information in a clean and visually appealing design.

## About the Project

The website is designed to represent my professional profile and provide visitors with information about my background, skills, and projects.

## Features

- Responsive Design
- Modern Teal, Black, and Blue Theme
- Sticky Navigation Bar
- Smooth Scrolling Navigation
- Hero Section with Introduction
- About Me Section
- Skills Section
- Projects Showcase
- Contact Information Section
- Interactive Buttons and Hover Effects
- Professional Portfolio Layout

## Technologies Used

- CSS3
